# 安全政策 | Security Policy

## 支持的版本

| 版本 | 支持状态 |
| --- | --- |
| 1.5.x | ✅ 支持 |
| 1.4.x | ✅ 安全更新 |
| < 1.4 | ❌ 不再支持 |

## 报告漏洞

如果你发现安全漏洞，请**不要**在公开的 Issue 中报告。

### 报告方式

发送邮件至：**adnaan.worker@gmail.com**

请包含以下信息：
- 漏洞描述
- 复现步骤
- 影响范围
- 可能的修复建议（如有）

### 响应时间

- 确认收到：48 小时内
- 初步评估：7 天内
- 修复发布：视严重程度，通常 30 天内

### 披露政策

- 我们会在修复发布后公开致谢（除非你希望匿名）
- 请在修复发布前保密漏洞详情

感谢你帮助保护 Adnify 用户的安全！
